Deskripsi Project
Project ini menggunakan dataset Mobile Price Classification dari Kaggle untuk memprediksi rentang harga smartphone berdasarkan spesifikasi teknisnya. Implementasi dilakukan menggunakan bahasa pemrograman Python di lingkungan Google Colab.




Dataset

Sumber: Mobile Price Classification - Kaggle.



Deskripsi: Dataset berisi spesifikasi fitur smartphone (RAM, Internal Memory, Baterai, dll) dan label target price_range (0-3).

Tahapan Project
Sesuai instruksi tugas, project ini mencakup lima tahapan utama:


Exploratory Data Analysis (EDA): Visualisasi korelasi fitur dan distribusi data.


Pre-processing Data: Pembersihan data dan pembagian dataset (Train/Test).


Feature Engineering: Standardisasi skala fitur menggunakan StandardScaler.


Training Model: Implementasi dan perbandingan model Random Forest dan Support Vector Machine (SVM).



Evaluasi & Analisis: Penilaian performa menggunakan Accuracy Score dan Confusion Matrix.
